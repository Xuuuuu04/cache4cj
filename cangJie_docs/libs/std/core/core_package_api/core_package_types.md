# 类型别名  
  
## type Byte
    
    public type Byte = UInt8
    

功能：[Byte](core_package_types.html#type-byte) 类型是内置类型 [UInt8](core_package_intrinsics.html#uint8) 的别名。

## type Int
    
    public type Int = Int64
    

功能：[Int](core_package_types.html#type-int) 类型是内置类型 [Int64](core_package_intrinsics.html#int64) 的别名。

## type UInt
    
    public type UInt = UInt64
    

功能：[UInt](core_package_types.html#type-uint) 类型是内置类型 [UInt64](core_package_intrinsics.html#uint64) 的别名。
